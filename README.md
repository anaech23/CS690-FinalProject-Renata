# Harlow Street Tattoo Studio Management System (v1.0.0)

An interactive, conflict-aware terminal application designed to manage tattoo studio schedules, clients, and artist leaves built on **.NET 10**.

## Quick Start
To run this application locally, ensure you have the **.NET 10 SDK** installed, then execute the following commands in your terminal:

```bash
cd HarlowStTattooStudio
dotnet run
